# Web Application Penetration Testing

## Overview
Performed basic web application security testing in a controlled environment using Burp Suite and Nmap.

## Tools Used
- Burp Suite
- Nmap

## Vulnerabilities Tested
- SQL Injection
- Cross-Site Scripting (XSS)

## Tasks Performed
- Intercepted HTTP requests and responses
- Tested input parameters with payloads
- Identified common web vulnerabilities
- Prepared vulnerability assessment reports

## Skills Gained
- Web Application Security Testing
- Vulnerability Assessment
- HTTP Analysis
- Report Writing
